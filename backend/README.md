# Exodus Backend

Backend implementation for the Exodus Code Challenge


# Functionality

Creates new accounts
Verifies passwords
Plaid Request


# Steps to initialize the backend

    1.) npm install
    
    2.) Add envirment variables to .env file 
            -Plaid API keys
            -MongoDB SRV
            -Desired Port
            -Regtest Data

    3.) npm run dev  OR npm start
